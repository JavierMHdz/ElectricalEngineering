%Angle needed in the entire problem
theta = 0:0.01:0.5;

%{
Using formula:
I/I0 = sin(x).^2/sin(phi).^2
where x = N*phi and phi = (pi*d*sin(theta))/lambda

since d = 3*lambda, the lambdas get cancelled. Leaving the following
equation:

phi = (pi*3*sin(theta))
%}

phi = (pi*3*sin(theta));

%Intensity = sin(x).^2/sin(phi).^2

%For N = 1
